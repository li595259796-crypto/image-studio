# Leo Image Studio 产品说明书

更新日期：2026-04-11

## 1. 产品概述

Leo Image Studio 是一款面向个人创作者、小型品牌团队和轻量运营场景的 AI 图像创作工具。它把“文生图”“上传图片后二次编辑”“作品沉淀管理”“账户与配额管理”集中到一个统一工作台里，目标是让用户用更少的步骤完成视觉草图、封面图、海报底图、风格头像与创意插画等任务。

当前产品提供中英文双语界面，默认语言为中文，支持登录后在账户设置中保存语言偏好。

## 2. 目标用户

- 需要快速出图的内容运营、独立创作者、自媒体作者
- 需要制作商品图、封面图、宣传图的电商或品牌从业者
- 需要用 AI 快速探索视觉方向的设计辅助角色
- 需要低门槛完成头像、插画、海报草图的普通用户

## 3. 核心价值

- 把常见图像创作任务收敛成少量清晰入口，降低上手门槛
- 生成、编辑、继续编辑、复制提示词形成闭环，减少重复输入
- 用图库沉淀历史成果，方便复用、筛选、收藏和下载
- 用账户体系和配额体系控制资源消耗，保证服务可持续运行

## 4. 产品功能

### 4.1 公开页面

- 首页：展示品牌、样例氛围图、产品主张，并引导注册或登录
- 登录页：使用邮箱和密码登录
- 注册页：创建新账户，成功后自动尝试登录
- 服务条款页与隐私政策页：承载基础法律与隐私信息

### 4.2 登录后工作台

- 创作页 `/generate`
  - 提供场景入口：商品图、封面视觉、海报底图、风格头像、创意插画、自由创作
  - 支持输入 prompt、选择宽高比、选择清晰度
  - 支持“Help me refine”对描述进行辅助完善
  - 生成成功后可下载、复制提示词、复制到创作页、继续编辑、重试

- 编辑页 `/edit`
  - 支持上传 1 到 2 张图片
  - 支持拖拽上传与客户端压缩
  - 支持从其他页面带入图片与 prompt，直接进入继续编辑流程
  - 编辑成功后可执行与生成页类似的后处理动作

- 作品库 `/gallery`
  - 按时间倒序查看历史图片
  - 支持分页
  - 支持收藏与取消收藏
  - 支持按时间范围和“仅收藏”过滤
  - 支持下载、删除、复制 prompt、继续编辑

- 账户设置 `/settings`
  - 修改昵称
  - 上传头像
  - 切换语言偏好并持久化到数据库
  - 修改密码

- 升级方案 `/upgrade`
  - 展示当前套餐和更高套餐能力
  - 当前版本的升级动作以邮件联系为主

### 4.3 配额机制

系统按用户维度限制每日和每月用量。默认免费计划为：

- 每日 10 次
- 每月 200 次

超出配额时，生成与编辑动作会被拦截，并返回对应提示。

### 4.4 多语言能力

- 支持 `zh` 与 `en`
- 未登录状态下语言偏好保存在浏览器本地
- 登录状态下语言偏好同时写入数据库
- 当前登录后唯一正式语言入口为“账户设置”

## 5. 典型使用流程

### 5.1 文生图

1. 用户登录后进入创作页
2. 选择一个创作场景，或直接进入自由创作
3. 输入提示词，选择宽高比和清晰度
4. 提交生成
5. 查看结果图并执行下载、继续编辑、复制 prompt 等动作

### 5.2 传图编辑

1. 用户进入编辑页
2. 上传 1 到 2 张原图
3. 输入编辑说明
4. 提交编辑
5. 查看结果图并继续二次处理

### 5.3 资产复用

1. 用户在作品库浏览历史图片
2. 通过过滤器缩小范围
3. 选择某张作品进行下载、收藏、复制 prompt 或继续编辑

## 6. 套餐与商业化现状

当前产品内置三档方案展示：

- Free
- Basic
- Pro

其中免费方案为系统默认生效方案。更高方案目前用于展示和商务承接，升级 CTA 会打开邮件联系。

## 7. 数据与用户资产

产品当前处理以下核心数据：

- 用户账户信息：邮箱、密码哈希、昵称、头像、语言偏好
- 图像资产：生成图、编辑图、Blob URL、大小、时间、收藏状态
- 使用记录：生成与编辑行为日志
- 任务记录：为未来异步队列保留的任务表

需要注意：

- 图片文件存放在 Vercel Blob
- 元数据、账户数据、配额日志存放在 Vercel Postgres
- AI 图像结果由外部图像 API 返回

## 8. 当前产品边界

- 当前主能力聚焦于图像生成与图像编辑，不包含团队协作、评论、审批流
- 当前商业化仍以套餐展示和邮件联系为主，未接入在线支付
- 当前线上主链路为同步生成和同步编辑
- 仓库中保留了异步任务队列实现，但当前产品并未把它作为默认用户链路启用

## 9. 运营与客服建议

- 当用户反馈“出图失败”时，优先确认是否为配额耗尽、网络超时或上游图像 API 波动
- 当用户反馈“语言未切换”时，引导其在账户设置中切换语言偏好
- 当用户反馈“历史作品缺失”时，优先核对数据库记录和 Blob 存储是否一致
- 当用户有升级需求时，当前统一通过 `support@image-studio.site` 承接

## 10. 产品成功标准

从当前实现看，产品的短期成功标准可以定义为：

- 用户能在登录后顺畅完成生成或编辑
- 生成结果可被沉淀到作品库并被复用
- 配额限制生效，避免资源失控
- 中英文切换体验稳定
- 登录、设置、图库、升级页构成完整基础产品闭环

## 11. 后续演进方向

- 优化创作页与编辑页的体验深度，提升专业感
- 重新启用或重构异步任务队列，解决长耗时请求体验问题
- 补全付费升级链路
- 增强作品库组织能力，如更多标签、搜索与批量操作
- 增加更完整的错误反馈和服务运行可观测性
